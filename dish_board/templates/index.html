רשימת הנתיבים של ה-API

Restaurants

תיאור נתיב מלא שיטה Header Body
קבלת כל המסעדות /info/ GET Content-Type: application/json לא נדרש
יצירת מסעדה חדשה /info/ POST Content-Type: application/json 
                        {
                            "name": "name",
                            "city": "city",
                            "address": "address"
                          }
קבלת מסעדה אחת /info/{id}/ GET Content-Type: application/json לא נדרש
עדכון פרטים למסעדה /info/{id}/ PATCH Content-Type: application/json 
                        {
                            "name": "name",
                            "city": "city",
                            "address": "address"
                          }
קבלת שעות פתיחה (חייבים להשתמש בפרמטר מזהה מסעדה) /opening-houers/?restaurant_id={restaurant_id} GET Content-Type: application/json לא נדרש
עדכון שעות פתיחה /opening-houers/{id}/ PATCH Content-Type: application/json 
                        {
                            "opening_time": "10:00",
                            "closing_time": "20:00",
                            "is_open": true
                          }

Menu

תיאור נתיב מלא שיטה Header Body
קבלת קטגוריות (חייבים להשתמש בפרמטר מזהה מסעדה) /category/?restaurant_id={restaurant_id} GET Content-Type: application/json לא נדרש
יצירת קטגוריה חדשה /category/ POST Content-Type: application/json 
                        {
                            "name": "firsts",
                            "restaurant": 1
                          }
עדכון קטגוריה קיימת /category/{id}/ PATCH Content-Type: application/json 
                        {
                            "name": "main",
                          }
מחיקת קטגוריה /category/{id}/ DELETE Content-Type: application/json לא נדרש
יצירת פריט חדש /item/ POST Content-Type: application/json 
                        {
                            "category": 3,
                            "name": "name",
                            "price": "42.00",
                            "description": "description"
                          }
קריאת פריט אחד /item/{id}/ GET Content-Type: application/json לא נדרש
עדכון פריט קיים /item/{id}/ PATCH Content-Type: application/json 
                        {
                            "name": "name",
                            "price": "42.00",
                            "description": "description"
                          }
מחיקת פריט /item/{id}/ DELETE Content-Type: application/json לא נדרש
